## Description

__Objetivo:__ Inserir tabelas de prós/contras, tabelas de ranking(colocações/justificativa/cta) e tabelas de especificações de produtos(dado/especificação) dentro de posts e páginas.

__Descrição:__ Criar 3 shortcodes. Um cria a tabela de pros e contras, o outro a tabela de ranking e o último tabelas de especificações. Os dados devem ser passados por atributos nos shortcodes.
